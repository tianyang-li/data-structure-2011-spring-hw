/***********************************************************
 * 本题中使用两个栈对题中过程进行模拟.
 * 一个栈用来模拟仓库的出入, 另一个栈用来存储当前仓库中的最大重量.
 * 有新的货物进入时 push 进仓库, 若重量大于等于当前最大重量则 push 进最大重量栈
 * 在货物出仓库时, 若当前出仓库的重量与最大重量相等则 pop 最大重量栈栈顶
 * 易得本算法时间复杂度为 O(n) 和空间复杂度为 O(n)
 * 该算法的正确性由栈 FIFO 的性质所保证, 所以不会出现最大重量栈出错的情况
 **********************************************************/
